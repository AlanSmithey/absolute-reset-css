# Modern CSS reset

A most useful set of rules for resetting styles in modern browsers!

## Target user agents:

* IE 11
* Edge 14+
* Firefox 52+
* Chrome/Chrome for Android 59+
* Safari/Mobile Safari 10+
* Opera 46+
* Android browser 4.4+

### Properties you need to reset yourself:

* Display
* Border
* Outline
* Font
* Text
* Color
